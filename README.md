## ğŸšš Installation
** This requires MongoDB to be installed and set up first **
<br>
1. Clone the codebase from the [repo](https://github.com/mpityo/budget-tracker).

3. Run the following command to install required packages:
```
npm install
```
4. Start the server:
```
npm run start
```
